É um sistema web com finalidade escolar para facilitar o compartilhamento de atividades, avisos e informações entre alunos, e organizar as tarefas de cada estudante, impulsionando o aluno a evoluir no seu percurso acadêmico que inicialmente será utilizado apenas para IFCE-Campus Iguatu, unidade areias. Os próprios alunos devem compartilhar essas atividades entre si.
